Load SAXS Data

Instructions for loading SAXS I(q) and P(r) data in SAXS-A-FOLD.

Overview

The "Load SAXS Data" tab allows users to upload or retrieve SAXS data files, including I(q) and P(r), for analysis. These files are essential for plotting and comparing experimental data with computational models.

Steps to Load SAXS Data

  1. Access the Load SAXS Data Tab: Navigate to the Load SAXS tab from the main interface.
  2. Upload or Retrieve Files:
    • SAXS I(q) File: Use the Browse local files button to upload a file from your computer, or the Browse server button to retrieve previously uploaded files from the server.
    • SAXS P(r) File: Follow the same steps as above to upload or retrieve a P(r) file.

    If no user-generated P(r) file is available, a link is provided to the BayesApp website for calculating P(r) from I(q) data.

  3. Submit the Data: Click the "Submit" button to upload and process the selected files. The data will be plotted as interactive graphs, with I(q) displayed in log-log mode.

Notes for Returning Users

  • If working on an existing project, previously uploaded I(q) and P(r) files will be available and do not need to be re-uploaded unless changes are required.
  • The uploaded file names will appear next to their respective Browse buttons after selection.

Data Visualization

Upon submission, the data is plotted as interactive graphs:

  • I(q): Displayed in log-log mode for clear visualization of the scattering data.
  • P(r): (If provided) Displayed alongside the I(q) graph to analyze the pair distribution function.
