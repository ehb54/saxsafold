Define Project

Instructions for defining projects in SAXS-A-FOLD.

Overview

The "Define Project" tab allows users to create a new project or access an existing one in SAXS-A-FOLD. Projects act as containers for data, settings, and results.

Steps to Define a Project

  1. Access the Define Project Tab: Navigate to the Define Project tab from the main interface.
  2. Enter Project Details:
    • Project Name: Provide a unique name for the project. This name is used to retrieve previously generated data.
    • Description: (Optional) Add a brief summary about the project.
  3. Submit the Project: Click the "Submit" button to save the project. For returning users, a dropdown menu will allow selecting previously defined projects by clicking in the Project Name field.

Tooltips and Additional Notes

  • Hovering over input fields or their labels will display tooltips with descriptions of their purpose.
  • For returning users, selecting a previously defined project automatically loads its data for further processing.
